
#  Sistema de Adoção de Animais com IA – Java Puro + (Swing)

Este é um sistema completo para **gerenciar adoções de animais** com um motor de **inteligência artificial simples**, desenvolvido 100% em **Java puro** com **interface gráfica (Swing)**.

O sistema permite:

- Cadastro de animais
- Simulação e cadastro de adotantes
- Recomendação do animal ideal com base no perfil do adotante
- Adoção e histórico completo de cada animal (saúde + adoção)
- Interface amigável e responsiva

---

## Funcionalidades

### Funcionais
- Cadastro de animais via interface
- Cadastro de adotantes (formulário)
- Recomendação por IA (baseada em regras)
- Realização da adoção
- Visualização do histórico completo do animal
- Interface gráfica em Java Swing

### Motor de Recomendação (IA)
O sistema utiliza um **motor de IA baseado em regras (Sistema Especialista)**, que analisa:

- Preferência de espécie
- Tipo de residência (casa/apartamento)
- Estilo de vida (ativo/calmo)
- Presença de crianças
- Tempo livre diário
- Necessidade de cuidados especiais

E gera uma **pontuação de compatibilidade** com cada animal disponível para adoção.
